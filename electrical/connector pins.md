# pin description of used connectors
### mains power in (`X1`)
`X1:1` : L1  
`X1:2` : PE  
`X1:3` : N  

### endstops connector (`X2`)
`X2:1` : top_es  
`X2:2` : DGND  
`X2:3` : bottom_es  
`X2:4` : DGND  

### door detection connector (`X3`)
`X3:1` : DGND  
`X3:2` : door_closed 

### start buttons connector (`X4`)
`X4:1` : start_pressed  
`X4:2` : DGND  

### linear actuator connector (`X5`)
`X5:1` : M+  
`X5:2` : M-
